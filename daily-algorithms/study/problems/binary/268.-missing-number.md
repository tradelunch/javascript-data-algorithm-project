# 268. Missing Number



{% embed url="https://leetcode.com/problems/missing-number/description/" %}



> time: O(N)

> space: O(1)

```jsx
/**
 * @param {number[]} nums
 * @return {number}
 */
var missingNumber = function(nums) {
    let sum = 0;
    let total = 0;
    
    for (let i = 0; i < nums.length; i++) {
        total += i + 1;
        sum += nums[i];
    }

    return total - sum;
};
```
